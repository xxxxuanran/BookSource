# Legado 3.0 BookSource

本仓库为适配 Legado 阅读 3.0 版书源仓库。

## 如何使用

推荐使用步骤：
1. 点击底栏 `订阅` Tab
2. 点击 `规则订阅` 选项
3. 点击 `添加`，类型为规则
4. 在 `Url` 处粘贴以下链接，名称随意
  `https://raw.githubusercontent.com/xxxxuanran/BookSource/refs/heads/master/bookSource.json`
5. 点击添加后的书源订阅，全选后确认导入

单次使用： [一键导入](legado://import/bookSource?src=https%3A%2F%2Fraw.githubusercontent.com%2Fxxxxuanran%2FBookSource%2Frefs%2Fheads%2Fmaster%2FbookSource.json)
